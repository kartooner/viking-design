# Viking Design Framework: The 5 Gates

Use this reference to determine if AI is the right tool for the job.

## The Gates

### Gate 0: Value
Is this worth building at all? Solve real pains, not shiny toys. If there's an easy workaround, value is low.

### Gate 1: Precision
Must it be 100% right? AI is for "close enough" (summaries, drafts, creativity). Use traditional code for math, taxes, and compliance.

### Gate 2: Logic
Can you write a checklist? If it's a list of If/Then rules, use automation/scripts. AI is for "fuzzy" tasks like sentiment or synthesis.

### Gate 3: Risk
What if it's wrong? High-risk (legal, safety, financial) requires a Human-in-the-Loop (HITL). AI shouldn't replace thinking.

### Gate 4: Context
Is the data digital and linked? AI needs good, searchable data. If info is in "shadow" spreadsheets or heads, the AI will fail.

---

## The Viking Decision
- **Fail any gate?** → Use a Database, Script, or Human instead.
- **Pass all gates?** → Proceed with Viking AI-First Design.
